# CampusHub

CampusHub is a long-term software engineering project focused on making university knowledge accessible, searchable, and community-driven.

Every year, students struggle with scattered information, endless group chats, and repeated questions. CampusHub aims to solve this by creating a centralized platform where students can ask questions, discover reliable answers from seniors, and easily search campus knowledge.

## Current Focus

The first version of CampusHub is a university Q&A platform with features such as:

* Student authentication
* Ask & answer questions
* Verified senior profiles
* Tags and categories
* Search functionality
* Upvotes for helpful answers

The goal is simple: **Make sure a student never has to ask the same question twice.**

## Future Vision

As the platform grows, CampusHub may expand to include mentorship, academic resources, internships, clubs, events, and other student-focused tools based on real user needs.
